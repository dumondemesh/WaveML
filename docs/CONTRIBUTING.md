# CONTRIBUTING.md — Вклад и практики

## Ветки и PR
- База релиза: `release/v0.2.0-rc1` (пример).  
- Фичи: короткие `feat/*`, `fix/*`, `chore/*`.  
- Conventional Commits: `feat:`, `fix:`, `chore:`, `docs:` …

## PR‑чек‑лист
- Лейблы `P?`/`I?`/`CI`/`docs`.
- Команды воспроизведения.
- Обновлённые/добавленные гейты (если нужно).
- Измерение/порог (если нужно).
- README/docs: 1 абзац «зачем/как».

## Репродьюсибилити
- Детерминизм по умолчанию: фиксируем порядок/seed, `RAYON_NUM_THREADS=1` в численных тестах.  
- Логи детерминированы: TimeWave‑ID, seed, hash‑паспорт.  
- Подписи `.wmpkg`, lock‑версии.
